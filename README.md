# 조직도 생성기 v5-1

텍스트 입력으로 SVG 조직도를 생성하는 웹 앱입니다.  
노드·가지 크기를 슬라이더 **및 숫자 직접 입력**으로 세밀하게 조절할 수 있습니다.

---

## ✅ 완성된 기능

### 📐 레이아웃 & 노드 크기 조절
- **레벨별 기본 노드 크기(L1~L4)**: 우측 패널 → 가로(너비) / 세로(높이) 직접 숫자 입력
  - `change` 이벤트 + Enter 키 지원으로 즉시 적용
- **개별 노드 가로 너비**: 줄기 클릭 → 팝업 → '이 노드 가로 너비 고정' 체크 후 슬라이더/숫자 입력
- **가지 세로 길이 (범위 20~500px)**: 줄기 클릭 → 팝업 → 슬라이더 드래그(실시간) 또는 숫자 입력 후 Enter/포커스 아웃

### 🎨 화면 커스텀 (우측 패널)
- 글꼴, 강조색, 캔버스 배경, 연결선 색상, 노드 색상 (대표/중간/하위/보조)
- **노드 둥글기·가로 간격·세로 간격·보조 X·Y·레벨1 X축**:
  - 슬라이더 ↔ 숫자 입력 **양방향 동기화** (슬라이더 드래그 → 숫자 즉시 반영, 숫자 입력 후 Enter/포커스 아웃 → 슬라이더 및 조직도 즉시 반영)
- 저장 / 뒤로 / 기본값 / 화면 커스텀 백업/복구

### 🎨 노드 개별 색상 변경 (NEW)
- **노드 클릭** → 색상 팝업 오픈
  - **헤더 색**: 노드 상단 색상 (컬러피커 + HEX 직접 입력)
  - **바디 배경**: 노드 하단 배경색 (컬러피커 + HEX 직접 입력)
  - **헤더 텍스트**: 이름 텍스트 색상 (컬러피커 + HEX 직접 입력)
- **이 노드 초기화**: 해당 노드만 테마 기본색으로 복원
- **전체 초기화**: 모든 노드 개별 색상 제거
- 변경 즉시 미리보기 반영
- **전체 백업/복구**에 노드별 색상 포함 (체크박스 선택 가능)

### 🌿 가지·줄기 개별 클릭 조절
- 모든 줄기(부모 수직줄기, 자식 수직줄기, 가로 수평바)에 **18~14px 히트 영역** 적용
- 클릭 팝업에 **타입 배지** (↓ 부모줄기 / ↓ 자식줄기 / ↔ 가로바) 표시
- 드래그 후 발생하는 **오발 클릭 차단** (5px 이상 이동 시 드래그로 판단)
- 단일 자식일 때도 가로바 히트 영역 생성

### 💾 백업/복구
| 위치 | 기능 |
|------|------|
| 툴바 → **전체 백업** | 5개 슬롯에 조직 텍스트+테마+레벨 크기+가지길이+노드너비 전체 저장/복원, 항목별 체크박스 선택, JSON 클립보드 복사 |
| 좌측 → **백업/복구** | 조직 텍스트만 백업/복원 |
| 우측 → **백업/복구** | 화면 커스텀(테마) 5슬롯 백업/복원 |

### 🖱️ 기타
- 마우스 드래그로 캔버스 이동 (5px 임계치 이후 캡처)
- 줌 슬라이더 (40~190%)
- SmartArt / Excel / 이미지 복사 / PNG 저장
- 레벨 미리보기 패널 (L1~L4 필터)
- 설명서 모달

## 🗂️ 입력 형식

```
# 들여쓰기 방식 (기본)
CEO | 회장 | 홍길동
  + 전략자문 |  |         ← + : 왼쪽 보조, - : 오른쪽 보조
  신재생사업 | 장 | 이태호
    Global Sourcing | 담당 | 김원희

# 기타: 불릿(- ), 화살표(>), 엑셀 열(Tab 구분)
```

## ⌨️ 단축키

| 키 | 동작 |
|----|------|
| `Tab` | 하위 들여쓰기 |
| `Shift+Tab` | 상위 내보내기 |
| `Ctrl+Enter` | 조직도 즉시 생성 |
| `Esc` | 팝업/모달 닫기 |
| `Enter` (숫자 입력란) | 즉시 적용 |

## ⚙️ 기술 구조

- 순수 HTML/CSS/JavaScript (외부 라이브러리 없음)
- `TreeLayout` 클래스: `assignMetrics` → `assignPositions` 순으로 레이아웃 계산
- `branchGapMap` (Map): nodeId → 세로 gap 오버라이드
- `nodeWidthMap` (Map): nodeId → 가로 너비 오버라이드
- `levelSizeMap` (Array[4]): 레벨별 기본 너비/높이
- SVG 인라인 렌더링 (`chartSvg`)
- localStorage를 통한 상태 저장 (`orgChart_v4`, `orgChart_v4_fullBackup`, `orgChart_v4_themeBackup`)
