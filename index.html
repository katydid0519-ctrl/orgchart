조직도 생성기 설명서

  1. Enter / Tab: 새 조직 추가 / 하위로 내려갑니다.
  2. Shift+Tab: 한 단계 위로 올라갑니다.
  3. + 조직명: 왼쪽 보조 노드, - 조직명: 오른쪽 보조 노드.
  4. 노드 클릭: 해당 노드의 헤더 색·바디 배경색·헤더 텍스트 색을 개별 변경합니다. 컬러피커 또는 HEX 직접 입력 모두 가능합니다.
  5. 세로 연결선 클릭: 해당 가지의 세로 길이와 노드 가로 너비를 개별 조절합니다. 슬라이더 또는 숫자 직접 입력 모두 가능합니다.
  6. 우측 패널 → 레벨별 기본 노드 크기: L1~L4 각 레벨의 가로(너비)/세로(높이) 기본값을 수치 입력으로 일괄 조절합니다. 빈칸이면 자동(텍스트 기반) 크기가 적용됩니다.
  7. SmartArt: Excel/PowerPoint SmartArt 텍스트 창에 붙여넣기 가능합니다.
  8. 마우스 드래그: 조직도 화면을 상하좌우로 이동합니다.
